Técnico de Nivel Superior en Informática con Mención en Ciberseguridad

Estudiantes Diurnos - V Semestre 2025

Perfil de Egreso

El egresado de Técnico Nivel Superior en Informática mención Ciberseguridad es capaz de programar y desarrollar sistemas informáticos, utilizando lenguajes de programación y diseño, y administrando servicios web. A su vez, gestiona bases de datos y evalúa la seguridad física y lógica mediante la exploración de vulnerabilidades del sistema, manteniendo un entorno de trabajo seguro, de acuerdo a estándares de desarrollo, seguridad y calidad de la industria.

Nuestro Sello Institucional

El egresado es capaz de actuar con un sentido ético, de responsabilidad cívica y ciudadana, respeto por las personas y el entorno, y contribuyendo a la innovación y emprendimiento, transformándose en un aporte real a las organizaciones en sus desafíos de hoy, comprometido con el medio ambiente y su sustentabilidad.

Proyectos de Titulación

Maximiliano Peñaloza

Pentesting Proactivo de Sitios Web
Aplicación de pruebas de seguridad sobre sitios web con el objetivo de identificar vulnerabilidades vigentes y reducir vectores de ataque, utilizando técnicas de pentesting ético.

Valentina Aguilera

Sitio Web para Tienda “Mundo Animal”
Desarrollo de una plataforma web orientada a la promoción y comercialización de artículos y alimentos para mascotas, incorporando funciones básicas de comercio electrónico.

Luis Cano

GymDex: Plataforma Comparativa de Gimnasios
Aplicación web enfocada en la comparación de gimnasios, permitiendo a los usuarios consultar información relevante como ubicación, servicios y precios, con el fin de apoyar su decisión de inscripción.

Gabriel Salinas

El Pollo Jugoso: Sitio Web con Integración de API
Desarrollo de un sitio web de presentación para un local de comida rápida, que consume datos desde una API externa para mostrar de forma dinámica su menú y productos.
